Resumen

Se pretende explicar de una manera concisa el funcionamiento de una calculadora online que permite la obtención de los valores azimut, elevación y datos adicionales que permiten aumentar la precisión del proceso.

Comunicacion con un módulo Wi-Fi ESP8266 a API del clima

Resumen

Se implementa una aplicación móvil que obtiene datos climáticos de una ubicación, obtenemos temperatura, longitud, latitud, etc. A su vez, esta aplicación se conecta a una base de datos y realiza una gráfica de temperaturas registradas.

Resumen
